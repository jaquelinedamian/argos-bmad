# SSD — Home Care 4.0 (Argos Saúde)

## Visão do Produto

O Home Care 4.0 é o núcleo inicial do ecossistema Argos Saúde.

## Objetivo do Sistema

Criar uma plataforma digital de cuidado contínuo com foco em fluxos, interfaces e organização funcional.

## Modelo de Identidade

O sistema adota um modelo de identidade unificada, em que um único usuário pode assumir múltiplos papéis, como administrador, paciente e prestador de serviço.
