# Argos BMAD

Estrutura multiagente baseada no BMAD Method para apoiar o desenvolvimento do ecossistema Argos Saúde.

## Objetivo

Apoiar a criação de fluxos, interfaces e organização funcional do sistema, com foco inicial no projeto acadêmico Home Care 4.0.

## Estrutura

- `docs` documentação do projeto
- `bmad` configuração dos agentes e contexto
- `assets` materiais de apoio

## Status

Em fase inicial de estruturação.
